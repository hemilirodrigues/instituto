# Instituto Barros — Website

Website institucional desenvolvido para clínica de medicina e estética avançada.

## ✨ Sobre o projeto

Este projeto foi criado com foco em:

- Design moderno e elegante
- Alta conversão de clientes
- Performance e responsividade
- Experiência premium (inspirada em grandes marcas)

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📱 Funcionalidades

- Layout responsivo
- Navegação fluida
- Seções institucionais (Sobre, Serviços, Valores)
- Botão direto para WhatsApp

## 🎯 Objetivo

Criar um site profissional para clínicas que desejam:

- Atrair mais clientes
- Transmitir autoridade
- Melhorar presença digital
